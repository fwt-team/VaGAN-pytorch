### Run VaGAN
__params:__  
-r Name of training run  
-n Number of epochs  
-s Name of dataset, default is minist  
-v version of model  
-ns control the switch of noise  

__example（GMM）:__  
python train.py -r vagan -n 500 -s mnist -v 1 -ns 0  
__example（SMM）:__  
python train_smm_v2.py -r vagan -n 500 -s mnist -v 2 -ns 0
